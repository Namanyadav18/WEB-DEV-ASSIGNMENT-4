# 📝 Flask To-Do List App

A full-stack dynamic To-Do List application built with **Flask**, **HTML5**, **CSS3**, and **Vanilla JavaScript**.

> Assignment 4 | Web Programming with Python & JavaScript Lab (SEC035)
> B.Tech Computer Science and Engineering | SOET

---

## 🚀 Features

- ✅ Add tasks with title, description, and priority (Low / Medium / High)
- ✏️ Inline edit tasks without page reload
- 🗑️ Delete tasks with smooth slide-out animation
- ☑️ Toggle task completion (strikethrough effect)
- 🔍 Filter tasks — All / Active / Completed
- 📊 Live task counter (Total / Active / Completed)
- 🎨 Priority colour-coded cards (Red = High, Orange = Medium, Green = Low)
- 📱 Fully responsive on mobile screens
- ⚡ All API calls use async/await Fetch — no page reloads

## 🔌 API Endpoints

| Method | Route | Description |
|--------|-------|-------------|
| GET | `/` | Render main page |
| GET | `/api/tasks` | Get all tasks |
| POST | `/api/tasks` | Create a new task |
| PUT | `/api/tasks/<id>` | Update a task |
| PATCH | `/api/tasks/<id>/toggle` | Toggle completion |
| DELETE | `/api/tasks/<id>` | Delete a task |
